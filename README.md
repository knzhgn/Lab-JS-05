# Lab 05: Modern JavaScript Syntax and Modular Architecture

## Structure
- `task1/` — Code Modernization (Lab 5.1): Legacy → ES6+ modules
- `task2/` — Modular Architecture with Dynamic Imports (Lab 5.2)

## How to Run

### Lab 5.1
```bash
cd task1
npm install  # (no deps needed, just sets type:module)
npm start
```

### Lab 5.2 (Node.js)
```bash
cd task2
npm start
```

### Lab 5.2 (Browser)
Serve the `task2/` directory with a local server:
```bash
npx serve task2/
# Then open http://localhost:3000
```
